片語¶

light up¶

  • 照亮 ; 使變亮 ; 使空間氣氛變歡樂
  • light > lit > lit

then and there¶

  • 當下
  • 同義
    • there and then
    • immediately

look forward to¶

  • 期待
  • 接V-ing

pass away¶

  • 過世 ; 死亡(委婉)
  • 同義
    • die
  • pass away from… 因…而過世
    • 同義
      • be died of

come around¶

  • (如往常一般)到來

keep up¶

  • 繼續下去

ever since¶

  • 從那時起
  • (Ever) since S + 過去式, S + 完成式

文法¶

S + have + O + p.p. …¶

  • S + have + O + p.p. 讓某人/某事被…
  • S + have + O + VR 要某人做…(使役)

wh- + to + VR¶

  • = wwh- + S + should/could/can + VR

S + make + O + VR¶

  • S + make + O + VR 使 某人/某物引起或導致某種反應
  • S + make + O + N 使某人/某物成為…
  • S + make + O + adj. 讓某人/某物…

S + will be + V-ing¶

  • 未來進行式
  • 表示在未來某時間點，依計畫或習慣，有個動作將正在進行中
