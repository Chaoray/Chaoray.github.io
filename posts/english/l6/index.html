文法¶

It is/was…that…¶

  • 強調句，又稱分裂句
  • 若強調主詞為人，可用who/whom

so + adj./adv. + that + S + V
such (+ a/an) + adj. + N + that + S + V¶

  • 如此…以致於

stand + N + high/tall
stand + adj.¶

  • 連綴動詞
    • + 數字 + 高度/長度單位 + high/tall
      • 高度為 ; 高達
      • stands 5 meters high = is 5 meters in height
    • + 形容詞
      • 說明建築物或物體處於某狀態
