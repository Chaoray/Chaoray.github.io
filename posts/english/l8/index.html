L8 大綱¶

英文哪來的大綱啦(老梗了啦)

單字¶

familiar¶

familiaradj.熟悉的事物/人 be familiar to 人
人 be familiar with 事物
familiarityn[U].熟悉familiarity with

urge¶

urgen[C].衝動have an urge to V 有一股衝動
resist an/the urge to V 忍住做某事的衝動
vt.催促urge sb. to V

srategy¶

srategyn[C].策略

visible¶

visibleadj.看得見的be visible to
反義：invisible (adj.) 隱形的
明顯的 ; 能注意的

purchase¶

purchasevt.購買同義：buy (vt.) 購買
n[C].購買make a purchase 購買

profitable¶

profitableadj.有利潤的profitable company 賺錢的公司
profitvi.獲得好處profit from... 從...獲得好處
n[C].利潤

similarly¶

similarlyadv.同樣地同義：likewise (adv.) 同樣地
similaradj.相似的A be similar to B (in sth.) (在某事物上)A和B相似
similarityn[C/U].相似性 ; 相似處A bear a striking similarty to B A和B十分相似

aim¶

aimvt.針對 ; 對象是aim...at... 把...針對...
vi.瞄準
n[C].目的 ; 目標

impulse¶

impulsen[C/U].衝動on impulse 在衝動之下
impulse buying 衝動購物

display¶

displayvt.展示 ; 陳列
顯露 ; 表現同義：{exhibit (vt.) 展示,exhibition (n[C].) 展覽}
n[C/U].陳列 ; 展覽on display

offer¶

offern[C].特價
出價 ; 報價
主動提議 ; 建議
vt.提供

customer¶

customern[C].顧客 ; 客戶regular customer 常客 ; 老主顧

advantage¶

advantagen[C/U].有利條件 ; 優勢it would be one's advantage to V 做...會對...有利
give sb. an advantage in... 讓某人在...有優勢
be at an advantage 處於優勢
反義：disadvantage (adj.) 不利條件 ; 劣勢

eventually¶

eventuallyadv.最後地同義：finally (adv.) 終於
eventualadj.最後的

trap¶

trapn[C].圈套fall into the trap of 落入...的圈套
陷阱
vt.困住trap > trapped > trapped

aware¶

awareadj.知道 ; 意識到連綴V/be aware of N
連綴V/be aware that S + V
as far as I aware (= as far as I know) 據我所知
awarenessn[U]. (sing.)意識 ; 認識連綴V/be aware of N

clever¶

cleveradj.聰明的 ; 聰穎的too clever by half 聰明過頭了

片語¶

be likely to¶

  • 有可能會
  • sb./sth be likely to VR
  • It is likely that S + V
  • 反義
    • be unlikely to 不太可能

within (…) reach¶

  • 在 (…) 可及範圍 ; 伸手便可處及的範圍
  • = within the reach of…

line up¶

  • 排隊
  • 同義
    • stand/wait in line

take advantage of¶

  • 利用
  • take advantage of 人 (占…的便宜 ; 欺騙)
  • take advantage of 事物 (利用或把握機會做某事)

set up¶

  • 設立 ; 設置

start off¶

  • 開始

文法¶

The + comparative + S + V…, the comparative + S + V…¶

  • 愈…愈…
  • 慣用語常省略主詞和動詞

S + V… + so that + S + aux. + VR¶

  • 為了要 ; 以便
  • 表目的的子句通常會有助動詞(will/can/would/could)

There is no need (+ for sb.) + to + VR…¶

  • 不需要，沒必要做某事
  • 沒有「It is no need之用法

If + S + V-es, (S + will/can/may +) VR…¶

  • if子句現在代未來，用來表示未來可能會成真的事
  • 主要子句可以是祈使句
