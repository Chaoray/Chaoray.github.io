R2 大綱¶

英文哪來的大綱啦

單字¶

positive¶

positiveadj.正面的反義：negative (adj.) 負面的 ; 否定的 ; 陰性的
積極的 ; 樂觀的
確信的 ; 肯定的
陽性的

signal¶

signaln[C].指示燈號traffic signal 交通號誌
the red stop signal 紅燈停的燈號
vt. vi.表達 ; 顯示

improve¶

improvevt.改善improve on sth. 改進某事物
improvementn[U].改善

intersection¶

intersectionn[C].十字路口(美)同義：junction (n[C].) 十字路口(英)

actually¶

actuallyadv.事實上同義：in fact,as a matter of fact,in reality 事實上 ; 實際上
actualadj.事際上的

¶

boothn[C].小隔間 ; 電話亭voting booth 投票亭
telephone booth 電話亭

pedestrian¶

pedestriann[C].行人

routine¶

routinen[C].日常事務 ; 例行公事daily routine 每日例行公事
adj.日常的 ; 例行的a routine check-up 例行檢查

experiment¶

experimentn[C].嘗試
實驗do/conduct/carry out experiment (on sth.) (在...)做實驗
vt. vi.試驗 ; 嘗試

twist¶

twistn[C].轉折 ; 轉變twists and turns 波折
vt.扭轉

conduct¶

conductvt.做(實驗、研究、調查等)
n[U].行為舉止 ; 表現

goal¶

goaln[C].目標achieve a goal 達成目標
同義：aim,objective (n[C].) 目標

escalator¶

escalatorn[C].電扶梯

ordinary¶

ordinaryadj.普通的相似：extraordinary (adj.) 特別的 ; 非凡的

motivate¶

motivatevt.激勵 ; 鼓勵motivate sb. to V
motivatingadj.激勵人心的同義：inspiring (adj.) 激勵人心的
motivationn[C/U].動機
n[U].激勵 ; 鼓勵

spark¶

sparkn[C].火花

activity¶

activityn[C].活動extracurricular activity (n[C]. pl.) 課外活動

promote¶

promotevt.促進
晉升promote sales 刺激銷量
promotionn[U].促進
n[C].晉升
促銷活動

片語¶

for the better¶

  • 好轉
  • change for the better
    • 好轉 ; 改善

take aaction¶

  • 採取行動
  • action (n[U].) 行動

put up¶

  • 設立 ; 設置 (告示 ; 海報)

thanks to¶

  • 幸虧
  • 同義
    • because of
    • owing to
    • on account of
    • as a result of

rather than¶

  • 而非
  • 置於句首接V-ing時，此時意同instead of

文法¶

by + V-ing/with + N¶

  • by + V-ing 藉由…動作
  • with + N 用…工具

The number of Ns + V-es…¶

  • the number of + 複數名詞 …的數目，後接單數動詞
  • a number of/numbers of + 複數名詞 一些… ; 很多…

S + be + that + S + V¶

  • 名詞子句放在beV後做主詞補語
